# Neo-Brutalism Blog Theme

一个采用新粗野主义（Neo-Brutalism）扁平设计风格的现代 WordPress 博客主题。

## 特色功能

### 设计风格
- **新粗野主义美学** - 粗边框、偏移阴影、明快的玫瑰粉色调
- **现代色彩系统** - 基于 oklch 色彩空间，色彩精准且易于调整
- **专业字体搭配** - Poppins（无衬线）、Lora（衬线标题）、Fira Code（代码）

### 布局与交互
- **响应式设计** - 完美适配桌面、平板和移动设备
- **粘性导航栏** - 随滚动固定在顶部，毛玻璃背景效果
- **文章卡片网格** - 精美的悬停动画和阴影效果
- **精选文章双栏卡片** - 大图展示重要内容

### 暗色模式
- **一键切换** - 明暗主题无缝切换
- **完整适配** - 所有组件和配色自动适配暗色模式

### 功能组件
- **侧边栏小工具** - 搜索框、分类列表、热门文章、标签云、订阅面板
- **固定分页导航** - 页面底部浮动显示，方便浏览
- **评论系统** - 完整的评论表单和评论展示
- **文章导航** - 上一篇/下一篇引导阅读

### 技术特性
- **纯 CSS 实现** - 无需额外依赖，性能优异
- **CSS 自定义属性** - 易于自定义主题颜色和样式
- **SEO 友好** - 语义化 HTML 结构
- **翻译就绪** - 支持多语言国际化

## 快速开始

**方法一**
1. 下载[releases](https://github.com/lappland22233/wp-/releases)的ZIP压缩包
2. 到wordpress后台的`外观`->`主题`->`安装主题`中上传ZIP压缩包
3. 激活主题
4. 开始撰写你的博客文章吧！

**方法二**
1. 下载主题文件夹
2. 将 `主题` 文件夹上传到 WordPress 的 `wp-content/themes/` 目录
3. 在 WordPress 后台「外观」→「主题」中激活主题
4. 开始撰写你的博客文章吧！

## 文件结构

```
主题/
├── assets/          # 资源文件夹
├── js/              # JavaScript 文件
├── template-parts/  # 模板部件
├── 404.php          # 404 页面
├── archive.php      # 归档页
├── comments.php     # 评论模板
├── footer.php       # 页脚
├── functions.php    # 主题函数
├── header.php       # 页头
├── index.php        # 首页模板
├── page.php         # 页面模板
├── search.php       # 搜索结果页
├── sidebar.php      # 侧边栏
├── single.php       # 文章页
└── style.css        # 主样式表
```

## 自定义颜色

编辑 `style.css` 开头的 CSS 自定义属性即可更改主题配色：

```css
:root {
  --color-primary: oklch(0.6209 0.1801 348.1385);  /* 主色调 */
  --color-secondary: oklch(0.8095 0.0694 198.1863); /* 次要色 */
  --color-accent: oklch(0.9195 0.0801 87.6670);     /* 强调色 */
  /* ... */
}
```
